# Cybertruck Omarchy Theme
Omarchy theme honoring Tesla’s Cybertruck. Dark backgrounds, grey / blue / white text. 

![Preview](preview.png)

**Cybertruck** is an Omarchy theme honoring Tesla's Cybertruck.

Inspired by the Cybertruck, it features:
- Blacks and stainless steel inpsired grays
- High-quality wallpapaers from Tesla and others that highlight the unique design of the Cybertruck

## Installation

```bash
omarchy-theme-install https://github.com/steve-lohmeyer/omarchy-cybertruck-theme
```
